# Asterisk Configuration

Dialplan configs, SIP/PJSIP registrations, and Asterisk module settings for the PBX.

## Key Files

- `extensions.conf` — Main dialplan routing logic (booth phone → game → upstairs phone)
- `pjsip.conf` — SIP endpoint registrations (Grandstream HT814 gateway, Yealink T46G)
- `asterisk.conf` — Core Asterisk settings
